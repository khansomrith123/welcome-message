## ✨ លក្ខណៈពិសេស

- 👋 សារស្វាគមន៍សមាជិកថ្មីដោយស្វ័យប្រវត្តិ
- 😢 សារចាកចេញដោយស្វ័យប្រវត្តិ  
- 🎭 ផ្តល់តួនាទីស្វ័យប្រវត្តិ
- 📝 គាំទ្រប្រើប្រាស់ placeholders
- 🔧 ការកំណត់ងាយស្រួលតាមរយៈ slash commands

## 📝 Placeholders ដែលប្រើបាន

| Placeholder | បកប្រែ |
|-------------|--------|
| `{user}` | លើកឡើងអ្នកប្រើប្រាស់ |
| `{user.name}` | ឈ្មោះអ្នកប្រើប្រាស់ |
| `{user.avatar}` | រូបតំណាងអ្នកប្រើប្រាស់ |
| `{server.name}` | ឈ្មោះម៉ាស៊ីនបម្រើ |
| `{membercount}` | ចំនួនសមាជិក |
| `{#channel}` | លើកឡើងឆានែល |
| `{:emoji}` | បង្ហាញអារម្មណ៍ |
